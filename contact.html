About the Page

For this project we used two primary sources of information. The first is from the National Cancer Institute to determine the "MET" value or "intensity" of the activity choices for our users. Although we only used ten activities in our page, the institutes' website lists upwards of one hundred different activities one can use to calculate calorie expenditure. The "MET" value is used in the final calculations to determine the calories burned during a strenuous (or not so strenuous) activity. This is done using the following formula.

Caloric Expenditure Formula

Calories Burned = MET VALUE x Weight(in Kilograms) x Time(in Hours)

The second primary source of information is from an equation known as the Harris-Benedict Formula. This is used to calculate a persons "BMR" or Basal Metabolic Rate. This formula is used on our "plan" page to give information about dietary needs to meet the users goal.

Harris-Benedict Formula

Women BMR = 655 + (4.35 x weight in pounds) + (4.7 x height in inches) - (4.7 x age in years)

Men BMR = 66 + (6.23 x weight in pounds) + (12.7 x height in inches) - (6.8 x age in year)
